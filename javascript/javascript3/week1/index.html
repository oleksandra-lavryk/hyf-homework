Fruityvice API

Fruityvice - it's an API that provides data about all kinds of fruit.

Currently the webservice consists of two functions:

  • receiving data for a specific fruit or all fruits
  • adding your own data

To receive the shown data you need to make HTTP GET call on the resource.

Call options:

  • /api/fruit/{id}
  • /api/fruit/{name}
  • /api/fruit/family/:family
  • /api/fruit/genus/:genus
  • /api/fruit/order/:order

API response with array of objects.

Object structure:

{
"genus": "Fragaria",
"name": "Strawberry",
"family": "Rosaceae",
"order": "Rosales",
"nutritions": {
"carbohydrates": 5.5,
"protein": 0,
"fat": 0.4,
"calories": 29,
"sugar": 5.4
}
}
